# Simulação de Fluido em Rust
Implementação de Simulação de Fluido em Rust utilizando a biblioteca glium para gráficos em OpenGL.

![a2 (1) (1)](https://github.com/user-attachments/assets/cf86de9a-f10f-4ce2-adfa-bf4b84829c6f)

### Como utilizar?

Basta clicar e arrastar o mouse que assim irá alterar o campo vetorial na direção que for movimentada.

### Como compilar?

Basta ter Rust instalado e rodar:

```cargo run```

Irá instalar as dependencias e rodar o programa e gerar um executavel dentro da pasta `target`.
